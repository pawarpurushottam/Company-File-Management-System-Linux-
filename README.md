# Company-File-Management-System-Linux-
In this project, a complete company directory structure was created using Linux commands. Different departments such as HR, Finance, IT, and Reports were organized into separate directories to simulate a real-world enterprise environment.

The project demonstrates essential Linux file management tasks including directory creation, file creation, editing files using Vim, copying and moving files, and generating reports using command redirectors. System information was also captured and stored in report files to simulate administrative reporting tasks.

Through this hands-on implementation, the project helps build practical skills in Linux file management, command-line operations, and basic system administration workflows used in real server environments.
